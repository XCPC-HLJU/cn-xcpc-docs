# cn-xcpc-docs

## 食用指南

netboot: 网络安装与配置选手机

domserver: Domjudge 服务端的安装与配置

judgehost: Domjudge 评测端的安装与配置

cds: ICPC Tools 中的 Contest Data Server 工具，是使用其他工具的核心 API

resolver: 滚榜工具

presentation: 大屏幕工具链

coachview: 远程显示选手机桌面与摄像头的工具

icpc-live: 直播 layout 工具

board: 外榜

docker: Docker 方式安装 domserver 和 judgehost
